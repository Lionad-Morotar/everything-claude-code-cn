# 构建与修复

逐步修复 TypeScript 构建和编译错误：

1. 运行构建：npm run build 或 pnpm build

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 对每个错误：
   - 显示错误上下文（前后各 5 行）
   - 解释问题
   - 提出修复方案
   - 应用修复
   - 重新运行构建
   - 验证错误是否解决

4. 停止条件：
   - 修复引入新错误
   - 同一错误在三次尝试后仍存在
   - 用户请求暂停

5. 显示摘要：
   - 已修复的错误
   - 剩余错误
   - 引入的新错误

为确保安全，一次只修复一个错误！
